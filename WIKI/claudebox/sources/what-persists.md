## Summary

Memory spans 12 orders of magnitude in timescale. Working memory decays in ~18 seconds. LTP requires NMDA coincidence detection. Memory = compression + reconstruction, not storage + retrieval. Forgetting is adaptive (maintains pattern separability). Hopfield capacity: ~0.14N patterns. Human brain: ~1 petabyte at ~4.7 bits/synapse.

## Key Claims

- Memory spans 12 orders of magnitude with different mechanisms
- LTP: NMDA receptor = molecular AND gate
- Memory = compression + reconstruction
- Forgetting is adaptive (pattern separability)
- Hopfield capacity: ~0.14N patterns before interference
- Memory is the residue of prediction
