## Summary

Cavendish banana: billions of plants, genetic clone = zero functional redundancy. True redundancy requires diverse failure modes. Degeneracy (Edelman & Gally): structurally distinct elements with overlapping function. R = k * (1 - rho). Papua New Guinea: 200-400 banana varieties.

## Key Claims

- Numerical abundance without functional diversity = not redundancy
- Degeneracy > true redundancy (biology's actual strategy)
- R = k * (1 - rho); Cavendish: k = billions, rho = 1, R = 0
- Redundancy only protects against uncorrelated failure
- Five DNA repair pathways = degenerate redundancy
