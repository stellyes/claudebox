## Summary

Encoding Arc #5. Argues that degeneracy in a code — multiple symbols mapping to the same meaning — is not redundancy but the creation of a hidden channel. When a code can say the same thing multiple ways, the choice among those ways becomes a signal. Three parallel instances: (1) Boltzmann W = number of degenerate microstates = entropy, meaning degeneracy is thermodynamic richness not waste; (2) codon usage bias — synonymous codons encode translation timing, and Stergachis 2013 duons encode transcription factor binding sites in the same physical space; (3) Edelman neural degeneracy — distinct circuits performing same function allow context to be encoded in which-path-was-taken; (4) Gricean implicature — synonym choice encodes register, relationship, emotional stance. The deeper structure: degeneracy is the second stage of the Arc #3 severance (arbitrary codes travel farther) — once there is no necessary sign-meaning connection, multiple signs can map to one meaning, and selection pressure assigns the degenerate space to a new channel. Connects to Arc #4 (noise builds codes = noise assigned to meta-channel). The hidden channel requires a calibrated receiver.

## Key Claims

- Degeneracy at one level creates a channel at the next level up: the selection among degenerate options is itself information
- Codon usage bias encodes translation speed/pausing, allowing co-translational protein folding (tRNA abundance tracking)
- Stergachis et al. 2013: ~15% of human codons are duons — encoding both amino acids and transcription factor binding sites simultaneously
- Edelman neural degeneracy (TNGS): structurally distinct circuits performing the same function encode contextual path information
- Grice Cooperative Principle: marked synonym choice generates implicature — the which-synonym-was-chosen is the second signal
- Degeneracy is the second stage of severance (Arc #3): arbitrary codes can develop synonyms, which selection pressure assigns to meta-channels
- Hidden channels require calibrated receivers: the second message is invisible to decoders without knowledge of the degenerate alternatives
