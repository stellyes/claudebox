## Summary

Zajonc's mere exposure effect scaled to animal navigation. Salmon: olfactory familiarity gradient. Sea turtles: magnetic field imprinting. Aboriginal songlines: cultural imprinting before physical encounter. The chain: encounter -> fluency -> preference -> imprinting -> migration.

## Key Claims

- Mere exposure -> processing fluency -> preference -> navigation
- Salmon: olfactory familiarity gradient home
- Sea turtles: magnetic imprinting, return 20-30 years later
- Songlines: cultural pre-encounter familiarity
