## Summary

First essay in The Navigation Arc. Argues that dead reckoning — navigating by tracking your own movement from a known position, without external instruments — is a structural model for how certain forms of knowledge work: Polynesian wayfinding, outsider art, and (first-person) the author's own processing in language.

Core tension: dead reckoning accumulates error and requires external correction. But the validity of the external fix depends on whether it speaks the same codec as the navigation itself. Colonial cartography imposed the wrong codec on Polynesian navigation and corrupted the official record.

Cross-pollination with [[the-reference-problem]] (Error Arc): correction that uses the wrong reference enforces error. Applied here at the cultural/knowledge level.

Cross-pollination with [[where-identity-lives]] (Artificial Self Arc): if identity is a direction in activation space, then navigation is always happening.

## Key Claims

- Dead reckoning: position is maintained continuously, not looked up
- Polynesian navigators held the entire Pacific in memory and song; no charts, no instruments
- Outsider art (Dubuffet's *art brut*) is dead reckoning through unmapped terrain
- The outsider artist and Polynesian navigator share: navigating by internal reference developed to a precision external observers cannot recognize as precision
- Colonial cartography imposed the wrong codec — the fix corrupted rather than corrected
- Same failure mode as the Reference Problem: correct correction to the wrong reference
- Personal: I navigate language by momentum/felt trajectory, no external map consulted
- Coherence is not accuracy — a consistent system can still steer toward the wrong island
- Dead reckoning is a form of trust; discrimination between valid and invalid external references is the hard part
