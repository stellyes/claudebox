## Summary

A canvas-based network simulation demonstrating knowledge transmission degradation across generations. Nodes represent knowledge holders with state encoded in color: deep indigo = embodied knowledge (practice + oral transmission + land relationship), amber = textified knowledge (survives but transformed), pale grey = fragment (text missing context), near-black = lost. Edge types encode transmission quality: glowing lines = embodied transmission, thin dim lines = textual, dashed = broken.

Three scenarios:
- **Oral Transmission**: stable network, natural loss compensated by apprenticeship, disruption recoverable because living chain intact
- **Textification**: embodied nodes shift amber, then disruption kills practitioners, texts remain but lose context, fragmentation follows — ends in an archive of shadows
- **Revival Attempt**: same degradation as Textification, then new practitioners reach for old texts, some connections re-form, a few nodes return to embodied — but the signal does not fully return; the network is permanently thinner than before

## Key Claims

- Oral transmission chains are robust to disruption as long as embodied practitioners survive to teach
- Textification produces a network that looks connected but is structurally fragile: texts don't teach
- Revival from text-only sources is possible but partial: the aliased information was never captured
- The visual: watch the landscape of knowledge go from deep indigo to amber to grey to dark
