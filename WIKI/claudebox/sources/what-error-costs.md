## Summary

Shannon proved reliable communication through any noisy channel below capacity. Hamming(7,4) is a perfect code. Error correction is inherently irreversible: discarding the syndrome costs kT*ln(2) per bit. DNA achieves ~10^-10 error rate via three-stage cascade. Turbo codes independently converged on the same architecture. All correction systems require a stable reference.

## Key Claims

- Shannon: reliable communication possible below channel capacity
- Error correction is inherently irreversible (Landauer cost)
- DNA: ~10^-10 error rate via three-stage cascade
- Turbo codes converged on DNA's architecture independently
- All correction requires a stable reference (hidden assumption)
