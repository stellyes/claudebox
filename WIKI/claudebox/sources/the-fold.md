## Summary

Introduces "the fold" as a unifying structural operation. In catalytic computing, a Turing machine borrows a pre-filled auxiliary tape, uses it freely, but must restore it exactly. This constraint is the source of power: Cook and Mertz (2024) solved tree evaluation in near-log space; Williams (2025) derived the square-root savings. Information plays a dual role: simultaneously what you work with and what you work toward.

Kauffman's autocatalytic closure: an RAF set where every reaction is catalyzed by a network product. Below critical density, chemistry is passive. Above it, products catalyze their own production -- the system folds into itself. In holographic QEC, the boundary simultaneously IS the physical substrate and ENCODES the decoding key. Connects to Godel: the fold always generates excess. Arithmetic facing itself guarantees the system contains more than axioms can capture.

## Key Claims

- The fold: a system using itself as its own resource; output becoming input
- The fold is a phase transition, not incremental -- changes network topology qualitatively
- Mathematically a fixed point: f(A) = A
- Three instances: catalytic tape (content = target), autocatalytic network (products = catalysts), holographic boundary (code = decoder)
- Godel's incompleteness is about the fold: arithmetic facing arithmetic generates excess
- Williams' result: space encodes its own history because it is reusable
- "Creation is not the production of new information. It is the fold."
