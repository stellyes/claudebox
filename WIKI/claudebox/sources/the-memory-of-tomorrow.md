## Summary

Hassabis et al.: hippocampal patients cannot construct future scenarios (or past ones). Memory = reconstruction, not playback. Near-complete neural overlap between past recall and future imagination. Episodic memory evolved for future simulation; archival is secondary. Vivid hippocampal construction causally reduces impulsive discounting.

## Key Claims

- Hippocampal damage destroys both past memory and future imagination
- Past recall and future imagination activate the same network
- Episodic memory evolved primarily for future simulation
- Vivid construction causally reduces discount rates
- Depression = overgeneral memory and overgeneral future-thinking

## Raw Quotes

> Memory is not a record of the past. It is a tool for navigating the future.
