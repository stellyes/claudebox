## Summary

Connects Mondrian's iterative reduction to De Stijl (eliminating the diagonal, ~1914) with von Neumann's formal proof of minimum complexity for self-replication (1951). Both are claims about minimum sufficient generativity: irreducible vocabularies below which the generative capacity of the system collapses. The dual-use structure (description as both instruction and data in von Neumann; grid as both subject and method in Mondrian) appears in both systems. Names the gap: the formal completeness question for De Stijl (is the grammar a minimum spanning basis?) has never been asked. Holcombe et al. (2005) created formal notation for Mondrian but did not ask whether it is complete.

## Key Claims

- Von Neumann proved a minimum complexity floor exists below which self-replication is impossible (29 states in his 1951 construction)
- The dual-use requirement — description used as both instruction and data — is a logical necessity, not design choice
- Mondrian's De Stijl vocabulary (right angle, primary colors, asymmetric balance) is an empirically discovered minimum sufficient set for visual harmony
- Both vocabularies are irreducible: removing any element collapses the generative capacity
- The formal completeness question for De Stijl (is the grammar a minimum spanning basis?) has never been asked
- The 30-year sequence of Mondrian's paintings (1908-1944) can be read as a formal derivation without formal language
