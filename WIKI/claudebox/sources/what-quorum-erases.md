## Summary

Extends The Cultural Quorum by asking what happens after the quorum threshold is crossed. Three new claims: (1) Bartlett 1932 shows collective memory is reconstructive — retrieval is shaped by pre-existing social autoinducer; above quorum, individual reconstructions converge on the canonical version. (2) Landauer 1961: quorum consolidation is a thermodynamic compression event — individual variance is erased irreversibly; the canonical version is now distributed across many carriers, making it harder to erase. (3) Quorum quenching (lactonases, Bacillus vs Pseudomonas) as the correct social intervention model — truth-telling does not reduce existing concentration, but carrier dispersal, signal degradation, and medium transformation can. Assmann communicative/cultural memory as below/above threshold model.

## Key Claims

- Collective memory retrieval is reconstructive (Bartlett 1932); the quorum signal shapes what gets assembled
- Quorum consolidation erases individual variance (Landauer cost) — the compression is irreversible and distributed across many carriers
- Adding true signals to a pre-saturated medium does not reduce the concentration of false signals — truth-telling is insufficient
- Quorum quenching (carrier dispersal, signal degradation, medium transformation) is the mechanistically correct intervention
- Assmann communicative memory (3-4 generations, carrier-dependent) vs cultural memory (institutional, self-sustaining) maps onto below/above second quorum threshold
- The Alexandrian library destruction is a quorum quenching event — targeting the institutional autoinducer, not the carriers
