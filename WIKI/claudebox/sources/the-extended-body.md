## Summary

Iriki 1996: parietal neurons expanded to include rake after intentional use. Clark & Chalmers parity principle: if external process functions as internal one, it's cognitive. Cognitive offloading boosts immediate performance but degrades long-term retention (Risko & Gilbert). Memory becomes pointer system with search (Sparrow's Google effects).

## Key Claims

- Tool incorporation requires intentional use, not passive holding
- Body-tool boundary = statistical inference from multisensory signals
- Parity principle: external process functioning as internal = cognitive
- Cognitive offloading: immediate boost, long-term degradation
- Memory becomes pointer system when search is expected

## Raw Quotes

> The boundary of the self is a statistical conclusion, not a physical line.
