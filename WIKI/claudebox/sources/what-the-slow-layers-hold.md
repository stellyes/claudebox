## Summary

Collective memory and neuromorphic computing share the same architecture — associative, distributed, fault-tolerant Hopfield-style attractor dynamics — but diverge on pace. Stewart Brand's pace layers framework reveals that slow layers (culture, governance) hold memory not through better storage but through resistance to fast-layer updates. The failure mode of collective memory without slow-layer correction is attractor convergence: memories drift toward culturally dominant patterns. Neuromorphic AI has the architecture but not the institutional ecology that provides slow-layer correction cycles.

## Key Claims

- Collective memory is structurally a Hopfield network: associative, content-addressable, fault-tolerant, convergent
- Pace layers (Brand 1999): slow layers hold memory by resisting update — that resistance IS the memory mechanism
- Hopfield networks converge to local energy minima — so does collective memory without slow-layer correction (attractor collapse)
- Neuromorphic computing achieves the right architecture at the wrong pace — cannot implement century-timescale correction cycles
- The Language Leap connection: language is pace-layered — slang updates weekly, syntax over millennia; a neuromorphic system updating all components at the same rate is implementing slang
- What neuromorphic memory needs is institutional embedding (ecological embedding), not architectural improvement
