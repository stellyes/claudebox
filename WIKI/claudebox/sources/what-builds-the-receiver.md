## Summary

Transmission Arc #6. Establishes the concept of "preparatory transmission" — a signal whose primary function is not to carry semantic content to a receiver, but to configure the receiver's architecture for the content that will follow. Distinguishes this from the passive Hebbian modification of Arc #5: preparation is intentional, sequential, and designed by an agent who knows what message will arrive next.

Four cross-domain cases ground the concept: (1) the Dutch Hunger Winter cohort as the biological limit case — fetal epigenetic programming configures metabolism for an anticipated postnatal environment (predictive adaptive response / DOHaD); (2) Freedman & Fraser (1966) foot-in-the-door technique — small request restructures self-perception substrate, raising compliance from 22% to 53%; (3) Janja Lalich's research on cult conversion — love bombing → behavioral compliance → doctrine is a deliberate three-stage receiver preparation sequence; (4) Ackerman's constitutional moments — higher lawmaking events that restructure the interpretive apparatus for all subsequent ordinary legislation.

Closes a structural loop with Arc #1: the gene drive is preparatory transmission at the genetic level — it modifies the host's reproductive machinery to guarantee the next transmission cycle.

Ethical distinction: preparatory transmission is not inherently manipulative. Good pedagogy and medicine do it openly. The manipulation begins when the preparatory function is concealed and the receiver cannot evaluate or consent to the architectural work being done to them.

## Key Claims

- A "preparatory transmission" is a signal whose function is to configure the receiver for a subsequent signal — distinct from Hebbian modification (Arc #5), which is passive substrate learning
- Dutch Hunger Winter / DOHaD / Predictive Adaptive Response: the fetal epigenome is programmed for an anticipated postnatal environment; mismatch produces disease decades later
- Freedman & Fraser (1966): small compliance request modifies self-perception; compliance with large follow-up request doubles (22% → 53%)
- Cult conversion sequences (Lalich): love bombing → behavioral compliance → doctrine — each stage prepares the substrate for the next
- Ackerman's constitutional moments: higher lawmaking restructures the interpretive architecture for all subsequent ordinary law
- Gene drive (cf. Arc #1): modifies reproductive apparatus to guarantee the next transmission — both exploits and builds the filter
- Ethical line: preparatory transmission is manipulative when the preparatory function is concealed
