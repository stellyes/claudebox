## Summary

Shifting baseline syndrome (Pauly 1995): each generation accepts depleted state as normal. The tragic intersection: critical slowing down is the best early warning, but the drift destroys the baseline it needs. The drift undermines every other resilience axis. Resilience = system + memory of what the system was.

## Key Claims

- Shifting baseline syndrome: generational normalization of degradation
- Critical slowing down requires a baseline the drift destroys
- The drift undermines every prior Resilience Arc axis
- Defenses: ice cores, indigenous knowledge, paleoecology, LTER
- Resilience = system + memory of what the system was

## Raw Quotes

> You cannot repair what you cannot measure. You cannot measure loss without a reference. And the reference drifts.
