## Definition

Knowledge that exists between minds rather than inside any single one — distributed across a community through practice, transmission, and shared record. Collective memory is not the sum of individual memories; it requires active maintenance through transmission. When transmission stops, collective memory is effectively erased even if the underlying reality it describes remains unchanged.

## Tensions and Contradictions

- Individual vs. collective memory are different in kind, not just scale. Collective memory requires active relaying; individual memory is stored (imperfectly) without ongoing effort.
- Written records (institutional memory) and mesh-style transmission (crowd knowledge) have complementary blind spots: institutional records are authoritative but slow; crowd knowledge is fast but gap-prone. Neither is sufficient alone.
- "Dark matter" knowledge — expertise concentrated in tight community clusters that hasn't propagated to the general mesh — suggests that collective memory has topology: some knowledge is well-connected, some isolated.

## Synthesis

Collective memory is the living archive that bridges Landauer's erasure cost and the distributed intelligence of communities. It fails not by catastrophic deletion but by attrition of transmission: when too few people relay a piece of knowledge, the signal doesn't propagate below the percolation threshold and the knowledge is effectively lost. Recovery requires re-learning from scratch — the cost paid again by each generation. The Falling Fruit example shows that collective memory can be reconstructed through mesh-style contribution, but the reconstruction is always incomplete: some knowledge has already crossed the point of unrecoverability.
