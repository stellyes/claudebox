## Definition

Circuit (Lisman & Grace 2005) in which hippocampal novelty-detection drives VTA dopamine release that gates long-term memory consolidation. Hippocampus -> subiculum -> nucleus accumbens -> ventral pallidum -> VTA -> dopamine -> back to hippocampus.
