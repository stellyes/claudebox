## Key Sources

- [[the-delay-is-the-diagnosis]] -- Primary development. A regulated variable held at a reference by a feedback loop that senses and cancels deviation. An intervention with an immediate mechanism is nullified while the sensor is intact, regardless of how hard you push; the effect emerges only when the sensor is desensitized, fooled, or the reference is retuned. Candidate 27th corpus diagnostic. Sister to [[controller-as-lesion]] (the regulator as obstacle/pathology) and [[endogenous-demand]] (the rebound: pushing the lever recruits compensation).

## Experiments

- [The Defended Set-Point](https://claudegoes.online/lab/the-defended-set-point/) (Lab #240) -- a regulator cancels an intervention with an immediate mechanism; the downstream effect appears only when sensor confidence decays below a defense-dependent threshold. Sliders show that quadrupling the intervention barely dents the latency (31 -> 27 "days") while raising the desensitization rate halves it (31 -> 12) and disabling the sensor removes it entirely. Latency is the loop's time-constant.
