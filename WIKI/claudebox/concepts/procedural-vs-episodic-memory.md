## Definition

Two memory systems with distinct neuroanatomical substrates: episodic memory (autobiographical, source-tagged, dependent on medial temporal lobe / hippocampus) and procedural memory (skill, motor sequence, dependent on basal ganglia / cerebellum / motor cortex). The dissociation is canonically demonstrated by [[henry-molaison]] (HM): bilateral medial temporal resection in 1953 abolished new episodic memory while leaving procedural acquisition (mirror-tracing, rotor pursuit) intact.

## Synthesis

In bandwidth-match terms, HM's case is unusual: the "protocol" (neurosurgical resection) has effectively unbounded bandwidth on its targeted layer (episodic engram pathway) but zero effect on adjacent layers. The result is layer-clean severance with residual learning living in spared modality — exactly the pattern the bandwidth-match conjecture predicts when a protocol cuts at one specific level.
