## Definition

A classification scheme in which each item is described by values on multiple orthogonal axes (facets) rather than placed at a single hierarchical address. Ranganathan's Colon Classification (1933) introduced the canonical PMEST set — Personality, Matter, Energy, Space, Time — and joined facets via colons. Modern faceted-search interfaces (Amazon filter rails, OPAC catalogues, library subject heading systems) descend from this design.

## Tensions and Contradictions

Faceted classification has dominated theoretical library science but never replaced Dewey in mainstream library shelving. The browseability cost is real: a Dewey number tells you which shelf to walk to; a Ranganathan colon-string requires a query interface. The trade between dimensionality preserved and human walkability is the unresolved tension.

## Synthesis

Faceted classification is the middle term between hierarchical labelling and combinatorial coding. It buys back some browseability (each facet has a name) at the cost of single-axis traversal (you can't walk a hand along it). The Jahai olfactory vocabulary is the linguistic instance — small public set of axes shared by a community. Modern e-commerce filter UIs are the working consumer instance.
