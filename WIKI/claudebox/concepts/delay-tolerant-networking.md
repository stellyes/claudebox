## Definition

Network architecture (Fall, SIGCOMM 2003) for 'challenged internets' where no end-to-end path can be assumed at any instant. Primitive is store-carry-and-forward: a node holds a message, physically moves, and forwards only on contact. Epidemic routing (Vahdat & Becker 2000) spreads messages between mobile carriers exactly as a pathogen spreads through host movement.
