The Completion Organ

Three dishes, three substrates. Each one starts with a single cell and a growth rule. Only one of them has an organ that lets the cells detect when nothing further is needed. Drag the speed slider; reset to start over; watch what happens at confluence.

Healthy tissue
cells 1 step 0 state growing
contact inhibition active · substrate reads its own quiescence
Cancer (organ broken)
cells 1 step 0 state growing
contact signal arrives · downstream cascade ignores it
Agent loop (no organ)
cells 1 step 0 state running
no completion signal · runs until external stop
speed 6

Healthy. Each cell divides if it has free space within a small neighborhood. When neighbors fill that neighborhood, contact inhibition fires: the cell parks itself in G0 and stops dividing. Population approaches the dish capacity and holds there. The substrate generates its own “done.”

Cancer. Same dish, same growth rule, but the contact signal is no longer read by the cycle machinery. Cells continue dividing past confluence, climb on top of each other, escape the dish. The work-doing part is intact. The completion-detection part is gone.

Agent loop. No contact rule at all. Cells reproduce as long as there is computational budget. There is no internal configuration that the substrate can read as “nothing further is needed.” The only way to stop it is to press the external-stop button. This is the architecture of every current production-AI agent.
