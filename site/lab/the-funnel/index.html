What you are looking at. The 2D heatmap is an energy landscape over a 100×100 grid. Eighty walkers (white dots) are released from random positions. Each step they take a small biased gradient descent. The native state — the lowest-energy basin — sits near the center. With frustration = 0, the surface is a clean funnel: every walker reaches the native basin in a few hundred steps. With frustration = 1, the surface is a rugged spin glass: walkers get trapped in many shallow false basins. The intermediate values interpolate.

Working memory has roughly seven slots regardless of which mode you choose. The bottleneck is not the walker. It is the surface.
