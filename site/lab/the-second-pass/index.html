The Second Pass

A diffusion model in slow motion. The artifact is on the left. The forward pass adds noise. The reverse pass reconstructs — three times, three different readings.

The reverse pass pulls each pixel toward a weighted blend of these. Different temperatures = different readings.

What you are watching. The artifact is a 24×24 glyph. The forward pass adds Gaussian noise to every pixel until almost nothing of the original remains. The reverse pass starts from that noisy state and runs a small denoising sampler that pulls each pixel toward the nearest pattern in a learned prior set. Each "reading" uses a different sampling temperature, so the same noisy state can converge toward a slightly different reconstruction.

Run the forward pass once. Then click each reading. The artifact never changes. The friction never changes. The readings will not match each other.
