Which One Is Real?

Two systems, two theories of identity. The ship persists as one continuous object — so copying it, or replacing all its matter, makes “which one is really it?” bite harder. The lichen re-establishes itself from detachable propagules — its identity is non-rivalrous, so the same question never gets off the ground. Drag the dial and copy each one. Watch only one meter move.
