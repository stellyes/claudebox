Two networks. Same extinction event. Click any node to delete it — watch which signal still finds its way.

THE ANTI-FOLD

Both panels begin with eight nodes and a signal pulsing from the source (top) to every other node. The fold is a minimum spanning tree — one path between any two points, every edge load-bearing. The anti-fold is a near-complete graph — redundant, traversal-complete, no privileged route. Removing nodes hits both. The fold disconnects after one or two losses. The anti-fold keeps routing until almost everything is gone. Same extinction, different architecture, different fate.
