Each dot is a lineage — a client training the shared genome against its own local environment. Every generation it drifts outward (local overfit), then germline reprogramming pulls it back toward the shared model. Drag the slider to set how hard the reset pulls. No reset: unbounded client drift, deleterious load climbing (dots redden). Full reset: every generation returns to the checkpoint.
