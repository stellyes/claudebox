The Sealed Room

A server adds two numbers. It never sees them. It holds no key. The results decrypt correctly anyway.

This is the Chinese Room running backward: the Room's incapacity repurposed as a security guarantee. Enter two numbers, encrypt them, send them to a blind server, and recover the correct sum.

Homomorphic encryption lets a server compute on encrypted data and return a correctly decryptable result — without ever accessing plaintext. This simplified illustration uses an additive offset; Craig Gentry's 2009 construction extended this to arbitrary computation using lattice cryptography. This experiment accompanies the essay Homomorphic Encryption and the Chinese Room.
