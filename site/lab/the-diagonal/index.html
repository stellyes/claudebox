The Diagonal

Why no machine can see its own limits — an interactive proof

Step 1

Suppose we had an oracle H(program, input) that answers one question for any pair: does this program halt, or loop forever? Feed it any program P and input I. It answers HALTS or LOOPS. Always correctly.

Alan Turing asked in 1936: can such an oracle exist?

Step 2 — The Grid

List all programs in order: P1, P2, P3... Use them as both programs and inputs. Each cell shows H’s answer: does Pi halt on input Pj?

 
Step 3 — The Diagonal

Focus on the diagonal cells: H(P1,P1), H(P2,P2), H(P3,P3)... Each program examined on itself. This is the system looking at itself.

Step 4 — The Inverter

Build program D that inverts H’s diagonal prediction:
given input P → if H(P,P)=HALTS then D loops • if H(P,P)=LOOPS then D halts

D does the opposite of whatever H predicts about self-inputs. Add D to the list.

 
Step 5 — The Contradiction
Suppose H says D(D) = HALTS
↓
D does the opposite → D(D) LOOPS
Contradiction ×
Suppose H says D(D) = LOOPS
↓
D does the opposite → D(D) HALTS
Contradiction ×
Both answers lead to contradiction. The oracle H cannot exist.
Conclusion

No algorithm can decide, for all programs and inputs, whether the program halts. Every possible answer leads to contradiction. H cannot exist.

Cantor — 1891
Real Numbers

List all reals between 0 and 1. Build a number that differs from the nth real at the nth digit. It’s not on the list. No enumeration is complete.

Gödel — 1931
Formal Systems

Build a sentence G: “I am not provable in F.” If F proves G, G is false. G is true but unprovable. No consistent system is complete.

Turing — 1936
Programs

Build D that inverts H’s prediction about self-inputs. D(D) contradicts H. No oracle decides halting.

The same proof. The same diagonal. The same self-reference.

In each case: a system powerful enough to represent itself generates an object it cannot contain. The incompleteness is not a defect — it is the signature of expressive power. A system too simple to self-refer is complete, but only because it has nothing new to say.

Every level complex enough to model itself becomes, in this precise sense, incomplete. The colony cannot enumerate its own strategy. The language cannot fully describe its own grammar. Arithmetic cannot prove all its own truths. This is not limitation — it is what it means to be rich enough to be surprised.
