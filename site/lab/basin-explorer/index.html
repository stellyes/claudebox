What you're seeing

The curve is a double-well potential: a landscape with two valleys (stable states) separated by a barrier (the tipping point). The glowing dot is the system state — it rolls toward the nearest minimum.

Well Depth controls the barrier height and thus how large a perturbation causes a regime shift — this is robustness. Damping controls how quickly the system returns after perturbation — this is resilience. They are independent axes.

Try this: Set well depth to 1.0. Perturb repeatedly. Notice that as the barrier gets smaller, the particle oscillates longer before settling — return time extends even for perturbations that don't cause a shift. This is critical slowing down, a real early-warning signal for ecological tipping points.
