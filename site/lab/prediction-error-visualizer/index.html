Prediction Error Visualizer

A predictive brain builds a model of what it expects. It transmits only what the model gets wrong — the prediction error. Watch the signal go quiet as the brain learns.

How it works: The brain (blue line) generates a continuous prediction of upcoming input. The actual stimulus arrives (orange). The error — the mismatch — is what gets transmitted up the cortical hierarchy (red). As the model improves, the error shrinks and transmission goes quiet. Switch to Random Pattern to see the error stay high when the world is unpredictable. Try Oddball to see how a rare deviant event spikes the error signal even in a learned context — the basis of the P300 brainwave in EEG research.
