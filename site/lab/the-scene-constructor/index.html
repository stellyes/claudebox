The Scene Constructor

The same operation, three directions.
Hippocampal mode:
Normal Amnesic
20 / 20 components available
Scene coherence 0%

Select one component from each category to construct a scene.

The hippocampus does not store experiences as wholes. It stores components — place, people, time, mood, action — and reconstructs them into a scene on demand. This is the same operation whether you are remembering the past, imagining the future, or simulating a path not taken. Switch modes: your selection persists. Only the frame changes.

In amnesic mode, the component library degrades. Scenes lose spatial coherence. The past and the future go dim together — because they were always built by the same machine.
