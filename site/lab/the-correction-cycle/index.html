How it works: Both sides run the same Hopfield network with three stored patterns. On the left, Pattern C is encoded with higher dominance (more repetitions in training), creating a wider attractor basin. On the right, a slow-layer correction cycle periodically re-balances all pattern weights. The key difference: minority queries on the left converge to the dominant pattern; on the right, slow-layer friction keeps them accurate.
