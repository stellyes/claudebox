What It Can Become

A brainless settling solver. It floods every corridor, then prunes to the one path that carries flow between source and goal — it becomes the answer. But it can only become a path to a goal it can physically reach. Change the problem and watch the limit appear.

Flooding…

Reachable: settling works — the body takes the shape of the solution. Sealed: the goal is real but unreachable, so there is no configuration to settle into (the becoming-constraint). Never existed: a representation-hungry problem — the absent leaves no trail to flow toward. Companion to What the Slime Mold Cannot Solve.
