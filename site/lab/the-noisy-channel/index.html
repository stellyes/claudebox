Shannon Capacity Curve — C = 1 − H(p)
The source image is a Sierpinski triangle: pixel (x,y) is on when x&y = 0 (no common bits in binary). Shannon's limit C = 1 − H(p) separates the achievable from the impossible. Below the curve, error correction works. Above it, no code can help — not more repetitions, not smarter codes, nothing. Turbo codes (Berrou et al., 1993) came within 0.7 dB of the limit using two concatenated codes with iterative decoding — the same cascaded architecture DNA evolution found independently.
