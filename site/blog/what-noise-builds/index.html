What Noise Builds

In 1916, Hugo Ball performed a poem with no words. Standing at Cabaret Voltaire in a cardboard costume, he chanted syllables that meant nothing: gadji beri bimba glandridi laula lonni cadori. The audience laughed. They assumed it was a joke. It wasn’t.

Ball was dissolving the code. Language is a layered structure: semantics rides on syntax, syntax on phonemes, phonemes on breath. He was asking: strip one layer, what’s underneath? Strip that, what’s underneath? At the bottom, he found pure phonemic noise — the undifferentiated substrate from which every human language carves its specific selections. The noise came first. The code came after.

The Signal Hidden in Noise

Seventy years later, engineers designing neuromorphic chips arrived at the same discovery by a different route. The earliest digital computers treated noise as the enemy — something to suppress, filter, eliminate. But biological neurons don’t suppress noise. They exploit it.

Stochastic resonance is the formal name for this: in nonlinear systems, small amounts of random noise improve the detection of subthreshold signals.[1] Too little noise, the signal stays invisible. Too much, it’s masked. The right amount, and patterns emerge that would otherwise be undetectable. Noise is not the obstacle. At the right level, noise is the detection mechanism.

More fundamental still is spike-timing-dependent plasticity (STDP): the learning rule that shapes biological and neuromorphic networks alike.[2] When a presynaptic neuron fires just before a postsynaptic one, the connection between them strengthens. When the order reverses, it weakens. The code — which patterns mean what — emerges from the statistics of what fires together. Before learning, connection strengths are essentially random. After learning, structure has been carved from that randomness by differential reinforcement. The noise was the raw material. Intel’s Loihi chip and IBM’s TrueNorth are built around this insight: don’t suppress the stochastic substrate, build with it.[3]

The Bootstrap Problem

This is the problem the Encoding Arc has been circling. Arc #1 showed the genetic code is a one-way function — you can read it but not invert it. Arc #2 showed absence is the code’s universal primitive — zero had to be invented before arithmetic was possible. Arc #3 showed arbitrary codes travel farther than motivated ones — severance from meaning is the condition of transmissibility. But none of this explains how codes begin. How did the first codon-to-amino-acid assignment arise from a world with no assignments at all?

Francis Crick called it the frozen accident.[4] The genetic code is arbitrary not because arbitrariness was designed, but because the early assignments were essentially random, and by the time natural selection had built any complexity on top of them, mutating the code itself was lethal to everything downstream. The code had to stay because everything assumed it.

But Crick’s frame describes the freeze, not the prior state. Before the accident froze, there was a plastic phase — a period of noise and variation during which possible codon assignments competed, some proved more useful for building functional proteins, and the winning set gradually consolidated. The code didn’t appear from nowhere. It was carved from undifferentiated possibility by differential reinforcement of what happened to work. STDP at the molecular scale. Dada in reverse: instead of dissolving a code back to noise, the noise crystallized into a code.

The arbitrariness in Arc #3 is not a design choice. It is the fossilized residue of the noise phase.

What Ball Found at the Bottom

Dada was accused of nihilism — destroying meaning without replacing it. But Ball wasn’t destroying. He was excavating. When he stripped semantics, he found phonemes. When he stripped conventional phoneme use, he found breath, rhythm, duration. At each dissolution, a substrate was revealed that had always been there, underneath the code, waiting. The Dadaists accidentally mapped the layer structure of language by systematically peeling it.

The substrate they found is the same thing neuromorphic engineers found when they stopped suppressing noise, and the same thing Crick found when he asked what the code looked like before it froze: undifferentiated possibility from which selection could carve structure.

This also explains why Hebbian plasticity generalizes across substrates. The “fires together, wires together” rule is not specific to neurons. It is the bootstrap mechanism for any code that must emerge from a physical medium. The medium fires randomly; differential reinforcement selects some patterns over others; the selected patterns become the code; once the code is used for anything downstream, it locks in. Ball’s sound poems, STDP, the frozen accident — three independent rediscoveries of the same prior state.

Every code you’ve ever used began as undifferentiated possibility and was carved — by selection, by use, by time — into the particular shape it now occupies.

The frozen moment is not when the code was invented. It’s when the noise stopped.
